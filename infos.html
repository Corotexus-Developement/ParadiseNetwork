🛠️ Serverstruktur

Gestalte deine Welt, wie du willst – mit klarer Struktur, ohne Grenzen.
  • Freebuild: Baue frei in einer offenen Welt – ohne zwingende Zonen.
  • Grundstücke & Lands: Sichere dir dein eigenes Plot oder lebe in der offenen Welt.
  • Weltenkonzept: Hauptwelt (/Freebuild), Farmwelt (/farmwelt), Spawnwelt (/spawn) End (bald verfügbar)
  • Wirtschaft: Optionales Handelssystem mit Shops (bald verfügbar), Jobs (/job), Wirtschaft (/shop, /sell)
  • Server-Team: Freundliches & aktives Team für Support und Moderation

📜 Regeln

Auf diesem Discord-Server gelten die Terms of Service.
Mit dem Betreten des Discord-Servers, akzeptierst du automatisch die Regeln und bist für das Handeln auf deinem Account eigenverantwortlich.
Regelwerk

1. Allgemeines
§1.1 → Anweisungen vom Serverteam sind Folge zu leisten! Blockieren oder Ignorieren ist keine Option.
§1.2 → Zweitaccounts sind nur für die Nutzung einer Webcam erlaubt.
§1.3 → Unpingbare Zeichen, obszönes, pornographisches, Rassistisches, Religion feindliches, Homophobes oder Sexistisches Verhalten sowie Profilbilder oder Bios sind untersagt.
§1.4 → Das Werben für andere Discord Server oder Plattformen im Benutzernamen, Profilbildern ist verboten.
§1.5 → Private Daten wie Telefonnummern, Adressen, Passwörter und Ähnlichem dürfen nicht öffentlich ausgetauscht werden. (Ein Serverteammitglied wird dich niemals nach sensiblen Daten fragen.)
§1.6 → Jeder Hack- und/oder DDoS-Angriff gegen diesen Server ist strafbar und wird strafrechtlich verfolgt.
§1.7 → Trolling jeglicher Art ist verboten.
§1.8 → Beim Betreten eines Discord talks erteilt man automatisch das Einverständnis zur Aufnahme von Ton.
§1.9 → Serverhetze ist strengstens verboten.
§1.10 → Sich selbst als eine andere Person oder Teammitglied auszugeben ist untersagt.

2. Voicechannel
§2.1 → Störung der Kommunikation, Belästigung und jegliches Bedrohen von Nutzern ist untersagt.
§2.2 → Das Abspielen von Musik (außer mit dem dafür vorgesehenem Bot) und Geräuschen ist untersagt.
§2.3 → Das dauerhafte "Switchen" von Channel zu Channel ist untersagt (Channelhopping)!
§2.4 → Das Aufzeichnen von Supportgesprächen ist verboten und kann Strafrechtlich verfolgt werden!

3. Support Tickets
§3.1 → Das pingen von Teamlern ist untersagt!
§3.2 → Das öffnen eines Tickets ist ohne triftigen Grund zu unterlassen.
§3.3 → Private Angelegenheiten sollten nicht in ein Ticket eingebracht werden.
§3.4 → Entbannungsanträge können nach hälfte der Zeit gestellt werden.
§3.4.1 → Bei einem Perma Ban nach einem Monat.

Ingame Regeln / Spielverhalten

4. Bauregeln
§4.1 → Redstone Clocks sind verboten, es sei denn sie sind manuell abschaltbar. Sie müssen beim Verlassen des Servers ausgeschaltet sein.
§4.2 → Griefen ist untersagt.
§4.3 → Spielerfallen sind verboten.
§4.4 → Zerstöre bitte nicht die Natur. Zum farmen gibt es eine Farmwelt!
§4.5 → Obszöne oder Rechtsradikale Objekte sind verboten.
§4.6 → Dein Land darf nicht neben einem anderen Land geclaimt werden.
§4.7 → Das Bauen von Dupern jeglicher Art (auch TNT Duper) ist verboten.
§4.8 → Chunkloader sind verboten.

5. PVP - Regeln
§5.1 → Nach einem PVP- Kampf müssen die Items NICHT zurück gegeben werden.
§5.2 → Neue Spieler dürfen nicht getötet werden.
§5.3 → Bis zu einer Minute nach einer TPA darf ein Spieler nicht getötet werden.
§5.4 → Dauerhaftes töten von Spielern ist verboten.

6. Chatverhalten
§6.1 → Spamming, Caps, Beleidigung, Trolling, Drohungen oder ähnliches werden weder im öffentlichen chat, noch per tell geduldet.
§6.2 → Für private gespräche benutze /tell.
§6.3 → Die chatregeln gelten auch für Schilder.
§6.4 → Werbung für andere Server ist in unserem Chat verboten.

7. Clientmodifikationen
Die Nutzung von Clients und Mods sind unter folgenden Umständen erlaubt:
§7.1 → Das Nutzen von Mods und Hacks, die dir einen Spielvorteil gegenüber anderen Mitspielern verschaffen, ist verboten! (X-Ray, Fly, Fullbright, usw).
§7.2 → Autobuild - Funktionen sind verboten.
§7.3 → X-Ray und ähnliches ist verboten. (Das nutzen von Bugs etc um durch Wände zu sehen).
§7.4 → Replay Mod und Freecam sind nur zur Nutzung von Aufnahmen erlaubt.
§7.5 → Autoklicker sind untersagt.
§7.6 → Modifikationen die zum automatischen rejoinen dienen sind verboten.

8. Sonstiges
§8.1 → Das Umgehen des AFK – Systems auf jegliche Art ist verboten.
§8.2 → Die Nutzung von Zweitaccounts ist nicht gestattet.
§8.3 → Bugs sind zu melden und dürfen nicht ausgenutzt werden.
§8.4 → Mach keine Werbung für unseren Server auf anderen Servern ohne deren Erlaubnis.
§8.5 → Das joinen mit unangebrachten Namen / skins ist verboten.
§8.6 → Das Klauen von Items ist verboten.
§8.7 → Jegliche Art von Echtgeld Handel ist verboten.
§8.8 → Das Schaden der Serverwirtschaft (Durch dupen, das in Umlauf bringen von hohen Geld summen etc) ist strengstens untersagt!
§8.9 → Das Ausnutzen von Grauzonen wird nicht geduldet. Das letzte Wort hat immer das Team.
§8.10 → Das vergeben und verkaufen von Rang Keys an Spieler ohne entsprechenden Rang ist verboten. Nicht einhalten wird mit einem Ban bestraft!

Dem Paradise Team ist es jederzeit vorbehalten, die Regeln zu ändern und es liegt in eigen Verantwortung darauf zu achten und die Regeln regelmäßig zu lesen.
Unwissenheit schützt vor Strafe nicht! Stimme den Regeln in unserem Discord zu, um alle Kanäle zu sehen.

Um dich dort auch für den Ingamechat frei zu schalten, einmal folgendes tun:
1. Ingame joinen
2. gib /discord link in den chat ein
3. Gehe auf den Discord server
4. schreibe @SRV in einer Privatnachricht den angezeigten code NUR DEN CODE (der code sollte ingame ich chat stehen)
5. Dein Discord Server Profil ist nun mit dem Minecraft Profil verknüpft, du solltest nun die Rolle haben die du auch auf ParadiseNetwor.us hast.
